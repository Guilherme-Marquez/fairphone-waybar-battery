# Fairphone 4 Waybar Battery Indicator

Scripts and configs to display a custom battery indicator on Waybar for Fairphone 4.

## Installation

1. Clone this repo:
   git clone https://github.com/yourusername/fairphone-waybar-battery.git
2. Enter the folder:
   cd fairphone-waybar-battery
3. Run the installer:
   ./install.sh

⚠️ **Use with caution:** This is tested only on a single device. I take no responsibility for any damage.
